# AI-Based Recommendation System

A Java-based recommendation system that suggests products to users based on ratings provided by other users.

## Features
- User-product rating management
- Recommendation generation
- Collection Framework usage
- HashMap-based data storage

## Technologies
- Java
- Eclipse IDE
- GitHub

## How to Run
1. Clone the repository
2. Open in Eclipse IDE
3. Run RecommendationSystem.java

## Future Enhancements
- Database integration
